Interactive scRNA-seq Analysis Workflow

This homepage serves as an overview page, briefly introducing the main steps of single-cell RNA sequencing analysis. Each step links to an individual tutorial page, such as QC, Normalization, Scaling, PCA, Clustering, UMAP, and downstream analyses. It is designed to function as the entry page for a GitHub Pages tutorial site.

Tutorial Map

Tutorial Steps

Each card links directly to the corresponding HTML page

🧪

QC

Step 1

Inspect nFeature, nCount, and percent.mt, and remove low-quality cells, doublets, and potential outliers.
⚖️

Normalization

Step 2

Correct sequencing depth differences between cells to improve comparability of expression levels.
✨

Variable Features

Step 3

Select genes with high biological variability as key input features for PCA and clustering.
📏

Scaling

Step 4

Perform Z-score standardization so genes are on comparable scales for PCA and downstream analysis.
📉

PCA

Step 5

Compress high-dimensional gene expression data into principal components while retaining key variation.
🧩

Clustering

Step 6

Construct a nearest-neighbor graph in PCA space to identify cell populations and subpopulations.
🗺️

UMAP

Step 7

Project high-dimensional data into 2D space to visualize relationships and distributions among cell groups.
🏷️

Annotation

Step 8

Assign cell type labels to clusters based on marker genes, biological knowledge, and literature.
🧾

Differential Expression

Step 9

Compare gene expression between clusters or conditions to identify markers and disease-related signals.
🔗

Integration

Advanced

Integrate multiple samples or batches to reduce batch effects while preserving biological differences.
💬

Cell-Cell Communication

Advanced

Analyze ligand–receptor interactions to explore signaling communication and microenvironment regulation.
🛤️

Trajectory / Pseudotime

Advanced

Infer cell state transitions and developmental trajectories, useful for studying differentiation or disease progression.
